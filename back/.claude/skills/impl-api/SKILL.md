## api 구현 지침
1. @./claude/skills/create-api/ 을 참고하여 api 인터페이스를 작성한다.
2. @./claude/skills/create-controller/ 을 참고하여 컨트롤러 클래스를 작성한다.
3. @./claude/skills/create-usecase/ 을 참고하여 유스케이스 클래스를 작성한다.
4. @./claude/skills/create-port/ 을 참고하여 포트 인터페이스를 작성한다.
5. @./claude/skills/create-adapter/ 을 참고하여 어댑터 클래스를 작성한다.
6. 각 단계에서 작성한 코드가 이전 단계에서 작성한 코드와 일관되도록 주의한다. 
7. 빌드를 통해 코드가 정상적으로 컴파일되는지 확인한다.
